About. Each algorithm produces a sharpness score in [0, 1]. The raw value is shown to its right. Cells in the heatmap are tinted by score: the more saturated, the sharper the line is according to that algorithm.

Storage. Saved lines persist in localStorage under sharpness-lab.lines. Seeds are dashed and reset by clicking reset seeds. Drawn lines are stored in canvas-internal coordinates (1600×700).

2-inch window. Computed from 96 px/in (CSS spec) scaled into canvas-internal coordinates using the canvas's current rendered size. The number of windows is ceil(L / 2in); in-between windows are added between each adjacent pair to smooth boundary effects.
